# Employee Attrition Prediction System

This project predicts the probability of an employee leaving an organization using machine learning and data analysis techniques.

## Features
- Data cleaning and preprocessing using Pandas & NumPy
- Exploratory Data Analysis (EDA)
- Logistic Regression model for attrition prediction
- Sentiment analysis on employee feedback (NLTK & TextBlob)
- Interactive Streamlit dashboard

## Dashboard Shows
- Individual employee attrition probability (%)
- Company-wide attrition rate
- Department-wise attrition analysis

## Tools Used
- Python
- Pandas, NumPy
- Scikit-learn
- NLTK, TextBlob
- Streamlit
- Matplotlib

## Dataset
- IBM HR Analytics dataset
- Custom HR dataset

## How to Run
- pip install -r requirements.txt
- streamlit run prediction_HR.py
- streamlit run prediction_IBM_dataset.py
- streamlit run feedback.py
